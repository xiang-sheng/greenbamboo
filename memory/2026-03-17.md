## 技术要点

### Cron 任务配置
- 使用 `openclaw cron add` 创建
- `--session isolated` 独立会话
- `--announce` 推送到聊天
- 需要指定具体 channel 和 to 地址

### 模型配置格式
```json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "qwen-portal/coder-model",
        "fallbacks": ["opencode/minimax-m2.5-free"]
      }
    }
  }
}
```
