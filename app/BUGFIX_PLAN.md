# Bug 修复计划

## 用户反馈问题

### 1. 存储模式选择交互问题 🔴
**问题**: 点击"本地存储"无反应，退出后却能直接进入
**原因**: `_selectLocalMode()` 调用后没有触发 UI 刷新
**修复**: 在 `setMode` 完成后调用 `setState` 刷新页面

### 2. 图标不一致 🟡
**问题**: App 启动图标与应用内图标不同
**原因**: 启动图标使用 `ic_launcher.xml`，应用内使用 `Icons.eco`
**修复**: 统一使用青竹图标

### 3. 预置数据问题 🔴
**问题**: 新安装就有预置数据（体重、睡眠等）
**原因**: `loadMetrics()` 从服务器获取数据，但本地模式时服务器不可用
**修复**: 
- 本地模式时显示默认指标模板
- 让用户自定义添加指标

### 4. 指标注册流程缺失 🔴
**问题**: 点击首页"+"提示"请先加载指标"，但无处注册指标
**原因**: 没有指标管理界面
**修复**: 
- 新增"指标管理"页面
- 首次进入引导用户创建指标

### 5. 记录页面冗余 🔴
**问题**: 用户希望直接点击指标卡片添加记录
**原因**: 当前设计有单独的"记录列表"页面
**修复**: 
- 移除底部导航的"记录"Tab
- 点击指标卡片直接弹出记录对话框

### 6. 历史补填功能 🟡
**问题**: 用户忘记记录时无法补填历史日期
**修复**: 在记录对话框中添加日期选择器
